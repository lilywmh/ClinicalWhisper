# Results

Output directory for all pipeline products. These files are generated by the notebooks and **should not be manually edited**.

## Expected Outputs

| File | Source | Description |
|------|--------|-------------|
| `stream_a_features.csv` | Stream A notebook | Per-participant acoustic features (f0, energy, pauses, etc.) |
| `stream_b_features.csv` | Stream B notebook | Per-subject NAcc BOLD values + anhedonia scores |
| `ablation_table.csv` | Fusion notebook | Cross-modal classification performance comparison |
| `roc_pr_comparison.png` | Fusion notebook | ROC + Precision-Recall curves overlay |
| `shap_*.png` | Fusion notebook | SHAP feature importance per stream |
| `confusion_matrices.png` | Fusion notebook | Held-out confusion matrices |
| `correlation_heatmap.png` | Fusion notebook | Feature correlation matrix |

## Note

This directory is gitignored (`Results/*.csv`, `Results/*.png`). Only aggregate outputs belong here — **never** raw audio, transcripts, or NIfTI files.
